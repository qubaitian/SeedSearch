# Seed Analyzer

warning: this mod may hurt you or help you, it depends on your perspective, please be aware of what it is.

fork from https://github.com/ForgottenArbiter/SeedSearch

you need a config like this

```json
{
  "ascensionLevel": 0,
  "playerClass": "IRONCLAD",
  "seed": "2PZ2CJWC6TPYD"
}
```
